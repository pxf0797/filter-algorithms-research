# 国内券商量化交易接口与港股通接入方案综合报告

> **版本**: v1.0 | **更新日期**: 2026-06-20 | **适用范围**: 个人量化投资者 / 小型量化工作室

---

## 0. 执行摘要

本报告整合三项研究成果，对国内券商量化交易生态进行全景式梳理，核心发现如下：

**发现一：国内券商量化接口已形成清晰的三层格局。** 顶层为迅投 QMT / 恒生 PTrade 两大券商定制平台（10~50 万门槛），中间层为 MiniQMT 开放式 API（开发者首选，10 万门槛），底层为中泰 XTP 极速柜台（<100μs 延迟，机构级）。另有聚宽/米筐/优矿/掘金四大第三方量化平台作为数据研究和策略原型环境。

**发现二：港股通可通过 QMT 实现量化交易，但需注意 7 项关键差异。** 包括必须使用 `.HKSH`/`.HKSZ` 后缀、T+2 交收对资金周转的影响、无涨跌幅限制带来的风控挑战、以及双边印花税 0.1% 带来的高交易成本（100 万 HKD 一次完整买卖总费用约 2,854 港元 + 汇率损耗约 1,380 元人民币）。

**发现三："股股通"并非独立平台。** 调查确认不存在名为"股股通"的券商或交易平台，最可能的指向是港股通（Stock Connect）的口误。同时，2026 年 5 月八部门联合整治已使富途/老虎/长桥等境外券商对内地用户全面受限。

**推荐路径：内地券商 + QMT/MiniQMT + 港股通** 是当前唯一合规且功能完整的量化交易方案。资金 10 万以上即可起步，50 万以上可使用全功能 QMT。

## 第 2 章：港股通量化交易方案

### 2.1 港股通交易规则对量化策略的影响

| 规则项 | 内容 | 对量化策略的影响 |
|:-------|:-----|:-----------------|
| **T+0 交易** | 当日买入可当日卖出，无次数限制 | 可做日内回转交易，但需考虑 T+2 交收的资金限制 |
| **T+2 交收** | T 日卖出，T+2 日资金才可用（买入 A 股）或可转出 | 当日卖出资金不可用于同日反向交易（与 A 股 T+1 不同） |
| **汇率双层机制** | 港币报价 + 人民币结算；参考汇率（±3%）+ 结算汇率（净额换汇） | 盘中预冻结资金比实际多约 3-5%，需预留资金冗余 |
| **交易日历** | 港股通仅在沪/深港三地均开市且结算系统正常时开放 | 年交易天数约 240-245 天（少于 A 股的 242 天），需过滤非共同交易日 |
| **额度限制** | 沪港通每日额度 420 亿 RMB，深港通 420 亿 RMB | 大额策略（千万级）可能在额度耗尽时无法交易 |
| **无涨跌幅限制** | 港股无涨跌停板，单日波动可能超 20-30% | 需设置止损，控制单笔风险敞口；波动率管理策略需更保守的参数 |
| **最小变动单位** | 随股价区间变化：0.001/0.005/0.01/0.02/0.05/0.1/0.2/0.5/1.0 港元 | 不同股价区间的 tick size 不同，影响限价单设置精度 |
| **港股通标的范围** | 恒生综合大型/中型/小型股指数成分，定期调整 | 策略需定期校验标的是否仍在可交易范围内，避免下单被拒 |
| **代码格式** | 5 位数字 + `.HKSH`（沪港通）或 `.HKSZ`（深港通） | QMT 中普通 `.HK` 后缀会被拒单，必须使用港股通专用后缀 |

### 2.2 支持港股通的券商量化平台

**QMT 的港股通支持状态：**

| 项目 | 状态 |
|:-----|:-----|
| 行情订阅 | 支持（需通过 xtdata.subscribe_quote + 港股通后缀） |
| 历史数据 | 支持（QMT 客户端「数据管理」中下载） |
| 下单交易 | 支持（限价单/增强限价盘） |
| 回测模拟 | 支持（需在 QMT 中正确配置港股通交易参数） |
| 行情档位 | **仅一档行情**，且为延迟行情（受港股行情授权限制） |
| 代码后缀 | `.HKSH`（沪港通）/ `.HKSZ`（深港通）/ `SGT`（部分新版 QMT） |

**港股通标的代码格式注意事项：**

```python
# 正确示例（QMT 中使用港股通后缀）
stock_codes_hk = [
    "00700.HKSH",   # 腾讯控股 — 沪港通通道
    "09988.HKSZ",   # 阿里巴巴 — 深港通通道
    "03690.HKSH",   # 美团 — 沪港通通道
    "00941.HKSH",   # 中国移动
    "02318.HKSZ",   # 中国平安
]

# 常见错误（会导致"无交易权限"错误）
wrong_codes = [
    "00700.HK",     # ❌ 普通港股代码，QMT 中不可用于港股通交易
    "0700.HK",      # ❌ 简写格式
]
```

### 2.3 港股通订单类型与时段规则

港股通的订单类型和交易时段与 A 股有显著差异，量化策略需针对性地适配：

**交易时段与对应订单类型：**

```
开市前时段 (9:00 - 9:30)
├── 输入买卖盘 (9:00-9:15)    → 竞价限价盘 | 可申报可撤单 | 价格 ≤ 前收盘 ±15%
├── 不可取消  (9:15-9:20)    → 竞价限价盘 | 可申报不可撤单
├── 随机对盘  (9:20-9:22)    → 不可撤单，系统随机对盘产生开盘价
└── 暂停时段  (9:22-9:30)    → 不可下单、不可撤单

持续交易时段
├── 早市 (9:30-12:00)        → 增强限价盘
└── 午市 (13:00-16:00)       → 增强限价盘

收市竞价时段 (16:00-16:10)
├── 参考价定价 (16:00-16:01) → 不可下单
├── 输入买卖盘 (16:01-16:06) → 竞价限价盘 | 价格 ≤ 参考价 ±5%
├── 不可取消   (16:06-16:08) → 不可撤单
└── 随机收市   (16:08-16:10) → 不可撤单，随机收市
```

**增强限价盘 vs 竞价限价盘对比：**

| 对比维度 | 增强限价盘 （增强限价盘） | 竞价限价盘 （竞价限价盘） |
|:---------|:-------------------------|:-------------------------|
| 适用时段 | 持续交易时段（9:30-16:00） | 开市前时段 + 收市竞价时段 |
| 配对范围 | 最多同时配对 10 条轮候队伍 | 按最终参考平衡价对盘 |
| 是否指定价格 | 是，但可与更优价格成交 | 是，有价格保护 |
| 未成交处理 | 转为普通限价盘继续排队 | 开市前：转限价盘进持续交易；收市竞价：自动取消 |
| 港股通是否可用 | ✅ 是 | ✅ 是 |

> **量化策略注意：** 港股通**不支持市价单**。需要模拟市价成交时，应使用增强限价盘并设置一个较远的价格（如卖一价向上 N 档），或使用 `LATEST_PRICE` 参数。持续交易时段结束后，未成交的增强限价盘会自动转入收市竞价（转为竞价限价盘），若仍未成交则自动取消。

### 3.1 调查结论

经过多轮搜索和交叉验证，**结论明确：不存在名为"股股通"的独立券商、交易平台或金融服务产品。**

### 3.2 最可能的实际指向

"股股通"极有可能是 **港股通（Stock Connect，即沪港通/深港通机制）** 的口误或混淆表达。

| 术语 | 英文 | 关系 |
|:-----|:-----|:-----|
| **港股通** | Stock Connect (Southbound) | 内地投资者通过上交所/深交所买卖港股的通道 |
| **沪港通** | Shanghai-HK Stock Connect | 2014 年开通，含沪股通 + 港股通 |
| **深港通** | Shenzhen-HK Stock Connect | 2016 年开通，含深股通 + 港股通 |
| ~~股股通~~ | ❌ | 无此平台 |

**混淆原因分析：**
- 用户可能将"沪港通/深港通"误记为"股股通"（三个字均带"股"字且结构相似）
- 或者是将"A 股通"（富途/长桥等券商为内地客户买卖 A 股的功能名）混用
- 或者是在交谈中出现的组合误记

### 3.3 名称相似平台排除

以下平台在功能或名称上与"股股通"有一定相似性，但均不相同：

| 平台 | 所属机构 | 功能 | 与"股股通"的差异 |
|:-----|:---------|:-----|:-----------------|

| **A 股通** | 富途/长桥 | 境外券商买卖 A 股的通道 | 功能方向相反（境外买 A 股，非内地买港股） |
| **股神通** | 部分财经媒体用语 | 非正式俗称 | 可能是"沪深股通"或"港股通"的另一简称混用 |
