## 1. 快速启动

### 方式一：Streamlit（推荐）

```bash
cd filter_research
pip install -r filter_app/requirements.txt
streamlit run filter_app/streamlit_app.py
```

浏览器自动打开 `http://localhost:8501`，侧边栏调节参数，关闭终端即停止服务。

### 方式二：Docker（生产部署）

```bash
cd filter_research
docker compose up --build -d
```

访问 `http://localhost:8501`。日志级别通过 `.env` 文件中 `LOGURU_LEVEL` 控制。

### 方式三：纯前端（零依赖）

```bash
cd filter_research
python3 -m http.server 8765
```

浏览器打开 `http://localhost:8765`，自动跳转到交互工具。

## 3. 操作步骤

### 3.1 选择市场和股票

- 侧边栏顶部选择市场（美股 / A股 / 港股）
- 输入股票代码（如 `AAPL`、`3690.HK`、`600519`）
- 首次输入自动拉取全部 8 个周期（1 分钟 / 5 分钟 / 15 分钟 / 60 分钟 / 日线 / 周线 / 月线 / 季线）的 K 线数据至本地 SQLite 数据库
- 后续访问从本地读取，可手动刷新

### 3.2 配置滤波方案

- 从预设下拉菜单选择已保存的方案，一键应用
- 或手动调节参数：选择 **滤波器** 类型，下方出现对应的参数滑块；勾选 **双滤波对比** 可额外选择第二个滤波器
- 支持的滤波器：SMA / EMA / WMA / ALMA / Savitzky-Golay / Kalman / Butterworth / Gaussian / Median / LOWESS（共 10 种）

### 3.3 分析图表

每个视图独立配置，支持不同周期和滤波器组合：

- **价格与滤波**：K 线 + 收盘价 + 滤波曲线（主滤波器/对比滤波器各一条）
- **残差图**：滤波输出与原始收盘价的差值
- **速度 v / 加速度 a**：滤波曲线的一阶/二阶数值梯度
- **施密特触发器**：三态信号（多 +1 / 观望 0 / 空 -1），自适应死区 ε\_t = k\_ε · max(σ\_t(v), σ\_min)
- **预测曲线**：抛物线拟合（抛物线方程）或二次多项式拟合（poly2），每段预测 N\_ext 个点
- **策略 PnL**：分段混合策略 — 保护期内启用止盈止损，趋势跟踪期仅止盈让利润奔跑；显示多空各笔交易、胜率、最大回撤
- **跨周期对齐**：高周期 PnL 事件标记映射到当前周期，对比两周期交易信号一致性
- **同向性判断**：高周期持仓时，本周期同向 PnL 才计入子图，反之维持不变

### 3.4 管理配置

- **保存预设**：当前全部参数命名保存，后续一键加载
- **更新/重命名/删除**：选中预设后可覆盖更新、改名或删除
- **JSON 配置导入/导出**：导出为 `filter_config.json` 文件，或拖入 JSON 文件恢复全部参数
- **数据库快照**：创建/恢复/删除 SQLite 快照，支持跨设备备份

### 3.5 数据管理

- **数据健康检查**：检测各周期数据完整性、缺失值、异常跳变
- **数据校验**：对比本地 DB 与 yfinance 源数据，发现历史修正或新增数据，一键更新
- **自动刷新**：开启后按设定间隔轮询 yfinance，自动更新 UI

## 4. 指标解读

图表区域顶部显示关键统计信息：

| 指标 | 含义 |
|------|------|
| **收盘价** | 最新价格（如 ¥150.23） |
| **σ** | 收盘价标准差，衡量波动幅度 |
| **平滑度** | 滤波输出的二阶差分平方和，越低越平滑 |
| **数据点数** | 当前视图显示的样本点数量 |

施密特触发器相关子图（在单个视图内多层叠加）：

| 子图 | 含义 |
|------|------|
| **速度 v** | 滤波曲线一阶梯度，反映趋势强弱 |
| **加速度 a / ±ε** | 二阶梯度 + 自适应死区（ε\_t = k\_ε · max(σ\_t(v), σ\_min)），σ(v) 为 EWMA 波动率估计 |
| **Sig 信号** | 三态输出：+1（多）/ 0（观望）/ -1（空），由速度和加速度跃迁阈值决定 |
| **PnL 收益** | 策略损益曲线，100 为初始本金；交易标记（三角形入场 / 圆形止盈 / 叉号止损） |
| **跨周期 PnL 参考** | 高周期策略事件映射到当前子图，便于对比两周期交易同步性 |

## 5. 典型工作流

### 场景 A：单股票多周期滤波对比

1. 在侧边栏选择市场并输入股票代码，等待数据拉取
2. 4 个视图分别配置不同周期（如日线 / 60分钟 / 15分钟 / 5分钟）
3. 各视图选择同一种或不同滤波器，观察不同时间尺度下的滤波效果
4. 启用双滤波对比，在同一视图叠加两种滤波器曲线

### 场景 B：施密特触发器 + 策略回测

1. 选择一个视图，确保开启 **施密特** 和 **预测**
2. 调节 k\_ε（灵敏度）、σ\_min（地板保护）、N\_EWMA（平滑周期）
3. 开启 **策略叠加**，设定止损阈值
4. 观察 Sig 信号 vs. PnL 曲线的对应关系，评估策略表现

### 场景 C：跨周期信号一致性验证

1. 日线视图开启策略叠加，生成高周期 PnL
2. 60 分钟视图开启 **显示高周期 PnL 参考**
3. 开启 **同向性判断**，观察两周期交易信号的同步性
4. 同向持仓区域以彩色高亮标记，反向区域置灰

### 场景 D：多参数蒙特卡洛扫描

1. 固定滤波器和周期
2. 使用预设方案快速切换参数组合
3. 通过 JSON 导出/导入批量对比不同参数下的平滑度、信号质量和 PnL

## 6. 常见问题

**Q: 切换股票后数据没有更新？**
A: 首次输入新代码会自动拉取。如需重新拉取，点击侧边栏 **刷新数据** 按钮，或开启 **自动刷新**。

**Q: 施密特信号没有显示（显示警告）？**
A: 数据点数 (N) 必须大于 N\_EWMA 参数，否则无法计算 EWMA 波动率。请增大 N 或降低 N\_EWMA。

**Q: 策略没有开仓（PnL 始终 100）？**
A: 策略需要有效的预测曲线和 Sig 信号。检查是否开启 **预测** 复选框，并确认滤波器输出无明显异常。

**Q: 高周期 PnL 参考子图为空？**
A: 需要先在高周期视图（如日线）开启并生成策略 PnL，再到当前视图开启 **显示高周期 PnL 参考**。

**Q: 如何导出当前图表？**
A: Plotly 图表右上角有工具栏，支持缩放、平移、框选缩放，以及下载为 PNG。

**Q: 如何批量对比多种参数组合？**
A: 使用预设方案保存不同参数组合，通过预设下拉菜单快速切换。也可以导出为 JSON 文件后在外部编辑再导入。

## 7. 技术说明

- 数据通过 yfinance API 实时获取，本地 SQLite 缓存，划分为 8 个周期独立存储
- 所有滤波算法基于 NumPy / SciPy 在后端实现，支持端点延拓缓解边界效应
- 施密特触发器基于 EWMA 波动率估计 + 速度和加速度双条件跃迁判据
- 预测曲线支持抛物线物理拟合和二次多项式拟合两种模式
- 策略 PnL 采用分段混合方案：保护期（止盈止损）+ 趋势跟踪期（仅止盈，让利润奔跑）
- 图表基于 Plotly 多子图布局（4-8 层自适应），暗色主题
- 配置管理支持预设（SQLite 持久化）+ JSON 导入/导出
