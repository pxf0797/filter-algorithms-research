# 滤波算法交互式对比分析工具 — 操作指南

## 1. 快速启动

### 方式一：Streamlit（推荐）

```bash
cd filter_research
pip install -r streamlit/requirements.txt
streamlit run streamlit/streamlit_app.py
```

浏览器自动打开 `http://localhost:8501`，侧边栏调节参数，关闭终端即停止服务。

### 方式二：纯前端（零依赖）

```bash
cd filter_research
python3 -m http.server 8765
```

浏览器打开 `http://localhost:8765`，自动跳转到交互工具。

## 3. 操作步骤

### 3.1 选择数据集

在左侧面板顶部的 **数据集** 下拉菜单中切换。共 5 种测试信号：

| 数据集 | 信号特征 | 噪声类型 | 适用验证场景 |
|--------|---------|---------|-------------|
| **正弦波 + 高斯噪声** | 5 Hz 正弦 | 高斯白噪声 σ=0.25 | 验证滤波器对周期性信号的保真度 |
| **阶跃信号** | 中点 0→1 跳变 | 高斯噪声 σ=0.15 | 测试瞬态响应和边缘保持能力 |
| **趋势 + 季节性** | 线性趋势 + 3 Hz 周期 | AR(1) 自相关噪声 | 验证趋势提取和季节性分离 |
| **脉冲信号** | 高斯脉冲 (t=1s, σ=20ms) | 噪声 σ=0.1 | 测试异常值/峰值保持能力 |
| **Chirp 扫频** | 1→20 Hz 线性扫频 | 极低噪声 σ=0.02 | 评估全频率范围的响应特性 |

### 3.2 切换数据长度

点击 **200 点** 或 **1000 点** 按钮切换数据长度：
- **200 点**：响应快速，适合快速探索参数
- **1000 点**：细节更丰富，适合精细分析

### 3.3 选择滤波器并调参

在 **滤波器** 下拉菜单中选择算法，下方动态出现对应的参数滑块：

| 滤波器 | 参数 | 关键调节技巧 |
|--------|------|-------------|
| **SMA** 简单移动平均 | 窗口大小 (3-101) | 窗口越大越平滑，延迟越大 |
| **EMA** 指数移动平均 | 跨度 (2-100) | 跨度小 = 更灵敏；跨度大 = 更平滑 |
| **WMA** 加权移动平均 | 窗口大小 (3-101) | 近点权重大，比 SMA 延迟略小 |
| **ALMA** Arnaud Legoux MA | 窗口 / 偏移 (0-1) / σ | 偏移量控制延迟-平滑平衡，0.85 为常用值 |
| **Savitzky-Golay** | 窗口 (5-101) / 阶数 (1-5) | 阶数 2-3 最常用；窗口约为特征宽度的 1-2 倍 |
| **Kalman** 卡尔曼 | Q 过程噪声 / R 测量噪声 | R 大 = 更信任模型；Q 大 = 更信任观测 |
| **Butterworth** 巴特沃斯 | 阶数 (1-8) / 截止频率 (Hz) | 阶数高 = 更陡峭截止；filtfilt 零相位 |
| **Gaussian** 高斯 | σ (0.5-20) | σ 越大越平滑；约等于 SMA 但无旁瓣 |
| **Median** 中值 | 窗口 (3-101) | 擅长去除脉冲噪声，保持边缘 |
| **LOWESS** 局部加权 | 数据比例 (0.01-0.5) | 比例小 = 更灵活但更慢；适合非线性趋势 |

参数修改后 **自动生效**（150ms 防抖），也可手动点击 **应用滤波** 按钮。

### 3.4 切换显示内容

使用复选框开关控制图表中曲线的显示/隐藏：
- **含噪信号**：灰色线，原始输入
- **干净信号**：黑色虚线，真实值参考
- **滤波输出**：彩色线，滤波器计算结果

### 3.5 自定义颜色

点击 **滤波输出颜色** 色块，弹出系统取色器，更换滤波输出曲线的颜色。

## 4. 指标解读

底部指标卡片实时显示 6 项评估指标：

| 指标 | 含义 | 越优方向 |
|------|------|---------|
| **MSE** | 均方误差 = 滤波输出与干净信号之差的平方均值 | ↓ 越小越好 |
| **RMSE** | 均方根误差，与原始信号同量纲 | ↓ 越小越好 |
| **MAE** | 平均绝对误差，对异常值不如 MSE 敏感 | ↓ 越小越好 |
| **SNR↑** | 信噪比改善 = 10·log₁₀(输入噪声方差/输出误差方差) | ↑ 越大越好 |
| **延迟** | 滤波输出相对输入的时间偏移（采样点数） | → 接近 0 最好 |
| **平滑度** | 滤波输出的二阶差分平方和，衡量曲线粗糙度 | ↓ 越小越平滑 |

> 绿色数值表示该指标表现优异，红色数值表示需要关注。

## 5. 典型工作流

### 场景 A：快速筛选最优滤波器

1. 选择目标数据集（如"正弦波 + 高斯噪声"）
2. 逐个切换滤波器，观察 MSE 和 SNR↑ 的变化
3. 在延迟和平滑度之间找到 Pareto 最优
4. 微调参数，确认最终选择

### 场景 B：对比两个滤波器

1. 先选择滤波器 A，调节参数至最优状态
2. 记住其指标数值（或截图）
3. 切换到滤波器 B，调节参数
4. 对比两者的 MSE、延迟、平滑度

### 场景 C：参数敏感性分析

1. 固定滤波器类型
2. 在浏览器中打开开发者工具的 Console
3. 逐步移动参数滑块，观察指标变化趋势
4. 找到参数变化的"临界点"（拐点即为最优参数）

### 场景 D：跨信号类型验证

1. 选定候选滤波器及参数
2. 逐一切换 5 种数据集
3. 观察滤波器在不同信号类型下的表现
4. 稳健的滤波器应在多种信号下都有不错的表现

## 6. 常见问题

**Q: 为什么切换滤波器后参数面板变了？**
A: 每种滤波器的参数不同，面板会根据滤波器定义自动渲染。

**Q: 图表出现 "⚠ 滤波输出包含大量无效值" 警告？**
A: 参数设置可能不合理（如窗口过大超出数据范围），调小窗口或增大 σ 即可。

**Q: 如何导出当前图表？**
A: Plotly 图表右上角有工具栏，支持缩放、平移、框选缩放，以及下载为 PNG。

**Q: 可以同时显示多个滤波器的结果吗？**
A: 当前版本一次只能显示一个滤波器。如需多滤波器对比，请参考 `tools/filter_comparison_tool.py` 生成静态对比图。

## 7. 技术说明

- 所有滤波算法均在浏览器端用 JavaScript 实现，无需后端服务
